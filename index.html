Tindak Balas Eksotermik

Tindak balas yang membebaskan haba ke persekitaran.

  • Suhu persekitaran meningkat.
  • Bekas tindak balas menjadi panas.
  • Jumlah kandungan tenaga hasil tindak balas lebih rendah daripada bahan tindak balas.
  • Perubahan entalpi, ΔH, bernilai negatif (-).

Tindak Balas Endotermik

Tindak balas yang menyerap haba dari persekitaran.

  • Suhu persekitaran menurun.
  • Bekas tindak balas menjadi sejuk.
  • Jumlah kandungan tenaga hasil tindak balas lebih tinggi daripada bahan tindak balas.
  • Perubahan entalpi, ΔH, bernilai positif (+).

Haba Pembakaran

Haba yang terbebas apabila 1 mol bahan dibakar dengan lengkap dalam oksigen berlebihan.

CH4(g) + 2O2(g) → CO2(g) + 2H2O(ce)     ΔH = -890 kJ mol-1

Nilai Bahan Api

Nilai bahan api ialah kuantiti haba yang terbebas apabila 1 g bahan api dibakar dengan lengkap dalam oksigen berlebihan. Unitnya ialah kJ g-1.

Contohnya, haba pembakaran metana (CH₄) ialah 890 kJ mol⁻¹. Jisim molar metana ialah 16 g mol⁻¹.

Nilai bahan api metana = (890 kJ mol⁻¹) / (16 g mol⁻¹) = 55.63 kJ g⁻¹.

Bagaimana Menentukan Bahan Api yang Baik?

Pemilihan bahan api yang baik bukan sahaja bergantung kepada nilai bahan apinya yang tinggi, tetapi juga beberapa faktor lain:

  • ✓ Kos Bahan Api: Bahan api yang lebih murah dan menjimatkan kos adalah lebih diutamakan.
  • ✓ Ketersediaan: Mudah didapati dan bekalannya tidak terhad.
  • ✓ Kesan terhadap Alam Sekitar: Pembakaran yang menghasilkan sedikit jelaga atau gas rumah hijau seperti karbon dioksida adalah lebih baik. Bahan api yang mesra alam tidak mencemarkan alam sekitar.
  • ✓ Teknologi: Sesetengah bahan api memerlukan teknologi canggih untuk digunakan, contohnya hidrogen.
  • ✓ Kemudahalihan: Mudah disimpan dan diangkut ke tempat yang diperlukan.

Aplikasi dalam Kehidupan Seharian

Aplikasi Tindak Balas Eksotermik

  • Pek Panas: Mengandungi bahan seperti kalsium klorida kontang atau natrium asetat. Apabila air ditambah atau butang di dalam pek ditekan, tindak balas eksotermik berlaku dan haba dibebaskan.
  • Tin Minuman Panas Kendiri: Menggunakan tindak balas antara kalsium oksida dan air untuk memanaskan minuman di dalamnya.

Aplikasi Tindak Balas Endotermik

  • Pek Sejuk: Mengandungi serbuk ammonium nitrat dan air. Apabila pek dipicit, serbuk larut dalam air dan menyerap haba dari persekitaran, menyebabkan pek menjadi sejuk. Digunakan untuk melegakan kecederaan sukan.
  • Membuat Kek: Penguraian natrium bikarbonat (serbuk penaik) apabila dipanaskan adalah tindak balas endotermik. Ia menyerap haba dan membebaskan gas karbon dioksida yang menaikkan adunan kek.

Pengiraan Haba Tindak Balas

Q = mcΔθ

Q = Kuantiti Haba (Joule, J)

m = Jisim Larutan (g)

c = Muatan Haba Tentu Air (4.2 J g-1 °C-1)

Δθ = Perubahan Suhu (°C)

Contoh Pengiraan Haba Peneutralan

Apabila 50 cm³ asid hidroklorik 1.0 mol dm⁻³ dicampurkan dengan 50 cm³ natrium hidroksida 1.0 mol dm⁻³, suhu campuran meningkat sebanyak 6.8 °C. Hitungkan haba yang dibebaskan dalam tindak balas ini.

Penyelesaian:

Jisim larutan, m = 50 cm³ + 50 cm³ = 100 g

Perubahan suhu, Δθ = 6.8 °C

Q = (100) × (4.2) × (6.8)

Q = 2856 J (atau 2.856 kJ)

Contoh Pengiraan Haba Penyesaran

Dalam satu eksperimen, serbuk zink berlebihan ditambah kepada 25 cm³ larutan kuprum(II) sulfat 0.2 mol dm⁻³. Suhu larutan meningkat sebanyak 5.0 °C. Hitungkan haba penyesaran kuprum oleh zink.

[Anggapan: Ketumpatan larutan = 1 g cm⁻³, Muatan haba tentu larutan, c = 4.2 J g⁻¹ °C⁻¹]

Penyelesaian:

Langkah 1: Hitung perubahan haba (Q)

Q = mcΔθ

Q = (25 g) × (4.2 J g⁻¹ °C⁻¹) × (5.0 °C)

Q = 525 J

Langkah 2: Hitung bilangan mol kuprum yang disesarkan

Bilangan mol CuSO₄ = (MV) / 1000

Bilangan mol CuSO₄ = (0.2 × 25) / 1000 = 0.005 mol

Daripada persamaan: Zn(p) + CuSO₄(ak) → ZnSO₄(ak) + Cu(p), 1 mol CuSO₄ menghasilkan 1 mol Cu. Jadi, 0.005 mol Cu disesarkan.

Langkah 3: Hitung haba penyesaran (ΔH)

ΔH = -Q / bilangan mol

ΔH = -525 J / 0.005 mol

ΔH = -105000 J mol⁻¹

ΔH = -105 kJ mol⁻¹
